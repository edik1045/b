## Installation:


- Download the backend.
- Extract the source code.
- Install the requirements ([NodeJS](https://nodejs.org/en/download/) is required).
```
    npm install
```
- Open ``run.bat``, then the backend should be running.

- Redirect to the server and enjoy.
